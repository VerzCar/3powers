# 3Powers

> **Spec is the law. Agents execute. Gates judge.**

An open, portable **judiciary kit** for spec-driven, agentic software delivery. It enforces a separation of powers — the spec as legislature, agents as executive, an independent oracle and deterministic gates as judiciary — across any model family, language toolchain, or LLM provider, with no required CI/CD platform.

**Status:** pre-implementation. Draft specification only.
