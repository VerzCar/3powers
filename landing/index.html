Open · Portable · Model-agnostic

Separation of powers for
agentic software delivery

When one model writes the spec, the code, the tests, and the review, validation becomes circular. 3Powers restores three independent branches — legislative, executive, and judicial — so you can trust what your agents ship.

Get started View on GitHub

Three independent branches

The spec is the law. Agents do the building. An independent judiciary decides whether the code matches the spec.

Legislative

The spec is the law

Every later stage answers to a versioned, EARS-form specification with an explicit risk tier and non-goals.

Executive

Agents do the building

Model-, language-, and provider-agnostic executive agents build against the spec, staying inside declared file scope.

Judicial

An independent judge

An independent oracle, a deterministic gate suite, and human review judge whether the code matches the spec.

An eight-stage lifecycle

Explicit human gates at spec approval and sign-off — everything else is automated and cheapest-first.

Get started

Install the engine and run the full lifecycle in one command. 

uv tool install ./engine
3pwr keygen
3pwr run "your intent here" --mode auto
